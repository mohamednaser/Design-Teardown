Person of the week

Marcos Cáceres (@marcosc) is a Staff Engineer at Mozilla. He has been working on web standards for over a decade and currently co-chairs the W3C’s Web Platform Incubator Community Group (WICG) and the Web Applications Working Group. Having developed for the web since 1996, Marcos is passionate about empowering web developers to build next-generation experiences and keeping the web safe, free, and open for all. Thank you for helping the web advance, dear Marcos!
